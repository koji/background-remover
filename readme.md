## how to run the code
1. clone this repo
2. install dependencies
```zsh
pip install -r requirements.txt
```
3. run the app
```zsh
python app.py
```


## how to host this on Huggingface space
https://huggingface.co/  
1. create a space 
2. select Gradio
3. push `app.py` and `requirements.txt`
